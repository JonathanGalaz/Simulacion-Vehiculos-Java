# 🚗 Simulación de Vehículos en Java

## Descripción

Proyecto académico desarrollado en Java para simular el movimiento simultáneo de tres vehículos utilizando programación concurrente mediante Threads.

El objetivo del proyecto fue comprender el funcionamiento de los hilos en Java, permitiendo que cada vehículo se ejecute de forma independiente.

## Tecnologías utilizadas

- Java
- Eclipse IDE
- Programación Orientada a Objetos
- Threads (Hilos)

## Funcionalidades

- Simulación de tres vehículos ejecutándose simultáneamente.
- Uso de clases e hilos independientes.
- Mensajes de ejecución en consola.
- Implementación de concurrencia en Java.
